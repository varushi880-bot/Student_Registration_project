#  Student Registration System

A simple Student Registration System built using Flask.

# Features

- Student Registration Form
- Form Validation
- Flask Backend
- Jinja2 Templates
- Responsive UI
- Success Page

#  Technologies Used

- Python
- Flask
- HTML5
- CSS3
- Jinja2

# Project Structure

Student_Registration_Project/

- app.py
- templates/
- static/
- requirements.txt

# Run the Project

1. Clone the repository

2. Install dependencies

pip install -r requirements.txt

3. Run

python app.py

4. Open

http://127.0.0.1:5000
